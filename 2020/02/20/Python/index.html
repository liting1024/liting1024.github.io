开始编码之前首先确保Mac的python处于最新版本
在终端中输入alias python=python 3替换最新版本
输入python –version查看目前的python版本

Before begin

保留字符

标识符

由英文数字下划线组成，且不以数字开头
单下划线开头表示不能直接访问的类属性
双下划线开头表示类的私有成员
双下划线开头和结尾是特殊方法专用标识 Eg：init( )代表类的构造函数

索引

sname：序列的名字

start：默认为0，开始位置

end：默认为序列长度，结束位置

step：默认为1，切片长度，隔step-1个

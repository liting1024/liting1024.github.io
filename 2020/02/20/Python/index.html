开始编码之前首先确保Mac的python处于最新版本
在终端中输入alias python=python3替换最新版本
输入python –version查看目前的python版本

Before begin

保留字符

缩进

不使用{ }
同行时可用分号
类比Swift语法

标识符

由英文数字下划线组成，且不以数字开头
单下划线开头表示不能直接访问的类属性
双下划线开头表示类的私有成员
双下划线开头和结尾是特殊方法专用标识 Eg：init( )代表类的构造函数

基本语法

输入输出

支持多个变量赋值

a = b = c = 1
a, b, c = 1, 2, “C”# 给多个对象分别赋值多个变量

运算符

** # 返回x的y次幂 Eg：10**2为100
// # 返回商的整数部分 Eg：9//2为4 -9//2为-5
is和is not # 判断俩个标识符是不是引用自一个对象，是则返回True
in和not in # 判断是不是成员

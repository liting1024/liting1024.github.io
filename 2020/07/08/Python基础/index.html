align：<内容做对齐；>内容右对齐；^内容居中；=内容右对齐，放在填充字符左侧，只对数字有效；需要配合width

#：二、八、十六进制显示0b/0o/0x的前缀
width：占用宽度
type：如下表

type 说明 type 说明
s 对字符串类型格式化 b 十进制转为二进制后格式化
d 十进制整数 o 十进制转为八进制后格式化

正则表达式

1、基础表达式

元字符
. 匹配除换行符以外的字符
\w 匹配字母、数字、下划线或汉字
\W 匹配除字母、数字、下划线或汉字以外的
\s 匹配单个空白符
\S 匹配除单个空白符外的字符
\d 匹配数字
\b 匹配单词的开始或结束，单词的分界通常是空格、标点、换行
^ 匹配字符串的开始
$ 匹配字符串的结束
限定符 举例
? 匹配前面的字符0次或1次 colou?r 匹配colour和color
+ 1次或多次 go+gle匹配gogle到goo…gle
* 0次或多次 go*gle匹配ggle到goo…gle
{n} n次 go{2}gle匹配google
{n,} 最少n次 go{2,}gle匹配google到goo…gle
{n,m} 匹配前面的字符最少n次，最多m次 employe{0,2}匹配employee，employ，employe

3、作为模式字符串使用

4、使用re模块实现

re.match(pattern, string, re.flags)

仅从起始位置匹配模式字符串

pattern：模式字符串
string：要匹配的字符串
flags：标志

flags
A或ASCII 对于\w、\W、lb、B、ld、D、\s和\S只进行ASCII匹配
I或IGNORECASE 执行不区分字母大小写的匹配
M或MULTILINE 将^和$用于包括整个字符串的开始和结尾的每一行（默认情况下，仅适用于整 个字符串的开始和结尾处）
S或DOTALL 使用（.） 字符匹配所有字符，包括换行符
X或VERBOSE 忽略模式字符串中未转义的空格和注释

re.search(pattern, string, re.flags)

搜索出第一个符合模式的字符串

re.findall(pattern, string, re.flags)

以列表形式返回所有符合的字符串

re.sub(pattern, repl, string, count, re.flags)

找到模式字符串并替换为repl

repl：替换的字符串

count：匹配后替换的最多次数

re.split(pattern, string, maxsplit, re.flags)

maxsplit：最大拆分次数

按照pattern的分隔符分隔为列表

文件和IO

创建打开、关闭文件

file = open(filename, mode, buffering, encoding=’utf-8’)

file：文件对象

filename：文件名，用’ ‘或” “括起来，如果打开的文件和当前文件同一目录下直接写文件名，否则需要写路径

mode：打开的模式，默认为r

buffering：缓冲区大小，默认为缓冲

encoding：文件编码方式，默认为GBK

file.close( )

刷新缓冲区中没有写入的信息再关闭文件

如果抛出异常则不能关闭文件

with expression as target:

expression：打开文件的函数

target：保存expression结果的变量

with-body：执行with后的语句，可以用pass代替

file.write(string)

写入字符

file.read(size)

读取size个字符，默认为全部字符，指针从文件开始读起

file.readline( ) 读取全部数据，但按行返回列表

file.seek(offset, whence)

移动指针

offset：移动的字符个数

whence：0表示从开头开始移动；1表示从当前位置；2表示从文章尾移动，默认为0

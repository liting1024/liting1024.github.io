Linux基本命令

文件操作

cd XXX 进入XXX文件夹（不写XXX时退出到主页）

ls 查看目录下的文件夹

mkdir XXX 创建新文件夹

rm -rf XXX 删除有文件的文件夹

终端操作

clear 清空终端屏幕

Tab键 自动推断补全内容

sudo 使用管理员身份（管理员创建的文件只能管理员修改）

sudo apt-get update 更新软件

sudo apt-get install tree 下载tree（输入tree可以使文件树状化）

还需要下载cat、touch、vim

vim XXX编辑

i 编辑模式

ESC -> :wq 保存

:q! 强制退出

其他操作

cat XXX查看

touch XXX 创建文件

python3 XXX.py 以python运行XXX.py

python

exit() 退出python

pip3 install gpiozero 下载gpio包

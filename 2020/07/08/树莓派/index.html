树莓派

Author: 舒克 Date: July 8, 2020  21:46:24 Category: Programme

笔记本电脑当作树莓派的显示器

修改静态地址

SSH远程连接树莓派

Linux基本命令

文件操作

cd XXX 进入XXX文件夹（不写XXX时退出到主页）

ls 查看目录下的文件夹

mkdir XXX 创建新文件夹

rm -rf XXX 删除有文件的文件夹

终端操作

clear 清空终端屏幕

Tab键 自动推断补全内容

sudo 使用管理员身份（管理员创建的文件只能管理员修改）

sudo apt-get update 更新软件

sudo apt-get install tree 下载tree（输入tree可以使文件树状化）

还需要下载cat、touch、vim

vim XXX编辑

i 编辑模式

ESC -> :wq 保存

:q! 强制退出

其他操作

cat XXX查看

touch XXX 创建文件

python3 XXX.py 以python运行XXX.py

python

exit() 退出python

pip3 install gpiozero 下载gpio包

利用Pycharm的ssh推送

iTerm2用ssh连接树莓派

在电脑本地和树莓派上同时创建同名文件夹

在Pycharm中打开文件夹，设置运行环境

在设置-网络中查看本机ip地址，输入Pycharm的SSH Configurations中

Host：本机地址

Username：pi

Password：树莓派的密码
Tools-Configuration 选择好path

Web path为默认的’/‘

Tools-Option-Create empty directories

GPIO

RPi.GPIO

gpiozero

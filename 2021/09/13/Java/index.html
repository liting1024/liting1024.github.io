输入

数据类型

基础数据类型：数值、字符、布尔

​ 数值：整数（byte、short、int、long）

​ 字符char

​ 布尔boolean：只能是true或false

引用数据类型：类Class、接口interface、数组

基本类型变量在栈，引用类型引用到堆（类似于指针）

Unicode编码每个字符占两位

不存在无符号数，可以用long处理

int转String

int i=12345;
String s=””;
第一种方法：s=i+””;
第二种方法：s=String.valueOf(i);

运算符

&&断路与：第一个操作数为假不判断第二个

||断路或：第一个操作数为真不判断第二个

逻辑与和或为& |

带符号右移>>

无符号右移>>>

0b可以定义二进制

=可以默认强制类型转换

数组

是引用类型

存在堆里

数组初始化

数组复制的常用方法有4种

  1. for循环，效率最低
  2. System.arraycopy() 效率最高
  3. Arrays.copyOf() 效率次于第二种方法
  4. Object.clone() 效率次于第二种和第三种

Eenhanced-For 增强for

只读式遍历

类、包和接口

类

构造方法（与类同名）没有的话系统产生一个默认构造方法（不嗲参数、方法体为空），也可以有多个构造方法

方法重载：多个方法名字相同，签名或者参数的个数、类型不同

和Cpp一样有this：可以调用另一个构造方法，但必须放在第一句，其他用法和Cpp一样

继承

extends 只继承一个直接父类但可以多重继承，自动继承父类的状态和行为，也可以修改（覆盖Override）、添加（参数相同）、重载（参数不同），同时继承超父类（SuperClass）

没有继承则自动继承java.lang.Object

字段的隐藏：重新定义一个从父类那里继承的域变量完全相同的变量

super：访问父类（的父类）的域和方法，可以在覆盖父类的方法同时又利用已定义好的父类方法
